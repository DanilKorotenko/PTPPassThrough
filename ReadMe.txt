### PTPPassThrough ###

===========================================================================
DESCRIPTION:

This is a sample camera client application that uses PTP pass-through APIs in ImageCaptureCore framework to communicate with a PTP camera. This demonstrates the simplicity of the ImageCaptureCore APIs to communicate with a PTP camera using PTP commands.

===========================================================================
BUILD REQUIREMENTS:

Mac OS X 10.6 or later

===========================================================================
RUNTIME REQUIREMENTS:
